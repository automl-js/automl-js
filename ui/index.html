Fully in-browser AutoML

Do not want to upload your data to a central site for processing? Want almost zero inference latency? Want to retrain models locally when new data is available? Want to run automated ML model selection on a dataset with zero setup? Then this page is for you.

On this page, you can run pattern recognition algorithms on your data. The data is processed locally in your browser. Just upload a CSV file, in which every row corresponds to a pair of inputs and outputs, and you are ready to go! An output is assumed to be values in last column, and the rest of values are inputs. Proper handling of different types of inputs as well as malformed or missing data is done automatically.
